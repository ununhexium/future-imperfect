A tool to install each python based tool in its own environment quickly and easily.

Installation

Install pipsi from PyPi using Python 3.

python3 -m pip install pipsi

Configuration

pipsi will not default to using the same virtual environment folder as the one configured for virtualenvwrapper. Assuming your already setup virtualenvwrapper: 

    alias pipsi="python3 -m pipsi --home $WORKON_HOME"

Usage

Let's install glances.

    pipsi install glances

It will now create a new virtualenv and add a symlink in $HOME/.local/bin/glances.

glances can now be started on the CLI.

If we want to run glances with its web server, we also need bottle. It was not installed by pipsi but that's an easy fix. 

    workon glances
pip install bottle
deactivate

glances can now be run in web server mode with

    glances -w
